# README

Can host locally by running rails server
Main is also connected to heroku, hosted remotely https://brightcity-api.herokuapp.com/letters 

Create local db:
Rails db:create
Rails db:migrate
Rails db:seed

Create remote db on heroku:
Open heroku console
Rails db:migrate
Rails db:seed
